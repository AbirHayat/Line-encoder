About Line Code Lab

Line Code Lab is an interactive visualizer for digital line coding techniques used in data communications and networking.

Supported Encodings

Unipolar NRZ 1 → High, 0 → Zero
Polar NRZ-L 1 → Positive, 0 → Negative
Polar NRZ-I 1 → Transition, 0 → No change
Polar RZ Returns to zero at mid-bit
Bipolar AMI Alternating polarity for 1s
Pseudoternary Alternating polarity for 0s
Manchester (IEEE 802.3) 1 → High-to-Low, 0 → Low-to-High
Differential Manchester 0 → Transition at start, always mid-transition
MLT-3 Cycles through -1, 0, +1 on 1s
2B1Q 2 bits per symbol, 4 levels
8B6T 8 bits mapped to 6 ternary symbols

Usage

  1. Enter a binary sequence (0s and 1s)
  2. Select one or more encoding schemes
  3. View the generated waveforms below
  4. Export individual or all waveforms as PNG
